1. Abstract

The Fire Alarm System aims to provide early fire detection using flame and temperature sensors. It triggers alarms and indicators to prevent damage and loss. Future enhancements may include integration with automated systems for suppression.

2. Purpose

To develop a reliable, responsive fire detection system using sensors and microcontrollers to ensure quick alerts and improve safety in homes and industries.

3. Scope

Includes sensor integration, alarm mechanisms, threshold calibration, microcontroller programming, real-time alerting, and testing across environments.

4. Block Diagram Description

  • Flame Sensor: Detects infrared from flames.
  • Temperature Sensor: Detects heat levels.
  • Microcontroller: Arduino or ESP8266 handles sensor data.
  • Alarm: LED and buzzer activated on fire detection.
  • Optional Wi-Fi: Sends remote notifications.

5. Code Snippet (Arduino)


const int ledpin = 13;
const int flamepin = A2;
const int buzpin = 11;
const int threshold = 500;

int flamesensvalue = 0;

void setup() {
  Serial.begin(9600);
  pinMode(ledpin, OUTPUT);
  pinMode(buzpin, OUTPUT);
}

void loop() {
  flamesensvalue = analogRead(flamepin);
  Serial.print("Flame Sensor Value: ");
  Serial.println(flamesensvalue);
  if (flamesensvalue <= threshold) {
    digitalWrite(ledpin, HIGH);
    tone(buzpin, 2000);
    delay(1000);
  } else {
    digitalWrite(ledpin, LOW);
    noTone(buzpin);
    delay(100);
  }
}
